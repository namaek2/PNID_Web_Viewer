# PNID_Web_Viewer
웹 기반 PNID 뷰어입니다.

패닝 : 마우스 드래그
줌 인/이웃 : 마우스 휠
새로운 사각형 추가 : shift + 마우스 드래그
사각형 크기, 각도 조절 및 이동 : 사각형 선택 후 조정

1. 폴더에 pull
2. node.js 14 설치
3. 아래의 터미널에서 다음 명령어들 설치
```bash
$ npm init -y
$ npm install webpack-cli --save-dev
$ npm install html-webpack-plugin
$ npm install webpack-dev-server -D
```
4. 다음 명령어로 실행
  ```bash
$ npm start
```
